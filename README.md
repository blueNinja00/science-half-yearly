# Science Half Yearly Quiz

A clean and modern Science Half Yearly Quiz website with Flashcards and Quiz Mode.

## 🚀 Features
- Flashcards for study with show/hide answers.
- Interactive quiz with multiple-choice, true/false, and short-answer questions.
- Instant feedback and progress bar.
- Clean, modern design.

## 🌐 Demo
Available at: https://yourusername.github.io/science-half-yearly-quiz/

## 🚀 Installation
1. Clone the repository: git clone https://github.com/yourusername/science-half-yearly-quiz.git
2. Open `index.html` in your browser.
## 📌 How to Host on GitHub Pages
1. Go to **Settings > Pages**.
2. Set **Source** to "Deploy from a branch".
3. Your site will be live at the provided URL
